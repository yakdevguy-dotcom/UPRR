# UPRR

Schedule app



Android only at this time sorry!!



This is a very crude and simple app to help the engineers and family members of the Union Pacific Rail Road see a visual representation of which days they have off. This was a side project in a language I had never attempted and was made to impress and help my dad. [Donations are Welcome!](https://buymeacoffee.com/yakdevguy)
